# vue-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
Warning: `app.js` will be Gzipped with .gz extension!


### Compiles and minifies for production
```
npm run build
```

Warning: `app.js` will be Gzipped with .gz extension!

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
